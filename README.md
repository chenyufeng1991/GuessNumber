# GuessNumber
&emsp;&emsp;&nbsp;这是一个在AppleWatch上实现的一个小型App，开发语言为Swift。是一个猜数字的游戏，屏幕上会出现不同数字的滚动，并能控制游戏的开始结束，让别人来猜数字。是不是很有意思。</br>
